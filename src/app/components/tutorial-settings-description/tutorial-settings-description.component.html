Settings

Here you can specify which forecasts you want to show and how they should be displayed.

Location Select for which location you want to show forecasts. We currently collect forecasts for Germany and Poland. There are also regional-level forecasts (German states or Polish voivodeships), but fewer teams make these available.
Target Forecasts are available for confirmed cases and deaths from COVID-19. For both targets forecasts can be shown on an incident (new cases or deaths) or cumulative (total cases or deaths) scale.
Forecast date / horizon Here you can select for which time point or for which horizon (1 through 4 weeks) you want to display forecasts, depending on your choice in “Show forecast by”. Note that you can also select a time point by clicking in the forecast graph below. By clicking on the play button you can start an animation.
Prediction interval Forecasts always come with some uncertainty. Forecasters can therefore specify intervals which shall cover the true value with 50% or 95% probability. Note that not all forecasters make such intervals available.
Shift models to Some modellers aim to forecast cases and deaths according to ECDC data, others according to data from Johns Hopkins University. For cumulative forecasts it can be helpful to align all forecasts with the same data to improve comparability. You can do this by selecting a truth data source here.
Show forecast by You can either look at one through four-week-ahead forecasts made at a specific time point or show all forecasts for a specific forecast horizon.
